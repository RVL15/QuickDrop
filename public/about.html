About the product

A clean, browser-based way to move files between devices.

QuickDrop uses a QR-linked session to keep the transfer flow simple: open the receiver on a laptop, scan from a phone, and send files directly through the browser with real-time updates.

Why QuickDrop

  • Fast setup with no account or app installation required.
  • Real-time upload feedback and instant receiver updates.
  • Designed for ad-hoc sharing between phone and laptop.

How it works

Each session gets a unique QR code and upload URL. The phone opens the upload page for that session, while the receiver listens for file events over Socket.IO and displays them immediately.

Core stack

Node.js Express Socket.IO Multer QRCode UUID HTML / CSS / JavaScript

Deployment note

QuickDrop runs locally by default and can also be exposed with a tunnel service such as ngrok when you want phone access from outside the local network.

Want a production-grade setup? Host QuickDrop behind HTTPS and use a stable public URL for the QR session generator.
