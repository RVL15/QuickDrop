About the product

A browser-based file transfer tool for laptop and mobile.

QuickDrop is built around one simple workflow: create a session on the laptop, scan the QR code from a phone, and transfer files in either direction with live browser updates.

Core idea

  • Create a QR-based session from the laptop in a few seconds.
  • Send files from phone to laptop or from laptop to phone.
  • See shared files appear instantly without refreshing the page.

How it works

QuickDrop generates a unique session, creates a QR code and upload link, and uses Socket.IO to keep both devices in sync as soon as a file is uploaded.

Why people use it

It is useful when you want a quick transfer without cables, Bluetooth pairing, or installing an app. It works well for documents, photos, videos, and everyday work files.

Core stack

Node.js Express Socket.IO Multer QRCode UUID HTML / CSS / JavaScript

Access and deployment

QuickDrop runs locally by default and can be exposed with ngrok when the phone is on a different network. This makes the same session reachable outside localhost during testing or demos.

For a production setup, host QuickDrop behind HTTPS and use a stable public URL for the QR session generator.
