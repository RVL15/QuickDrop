About QuickDrop

Fast, secure, browser-based file transfer via QR

QuickDrop enables quick, ad-hoc file transfers between devices using a simple QR-code workflow — no account required, no plugins. The receiver generates a short-lived session and QR code; the sender scans the QR on their phone to upload files directly to the receiver.

Why QuickDrop?

  • Simple: Open the receiver page, scan, and send — no sign-in or setup.
  • Real-time: Files appear instantly on the receiving device using Socket.IO.
  • Temporary sessions: Transfers are short-lived to reduce exposure.

Deployment & Access

QuickDrop can be run locally or deployed behind a public tunnel (for example ngrok) when you need to receive files from mobile devices over the internet. For production use, host QuickDrop on a secure, trusted server and ensure HTTPS is enabled.
